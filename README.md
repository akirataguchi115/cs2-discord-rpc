# Free and open-source Discord Rich Presence for CS:GO

---
## How to use:
1. Download the latest release from this repository
2. Download node modules:
    ```shell
    npm install
    ```
3. Run code:
    ```shell
    node app.js
    ```
4. Open CS:GO and enjoy!
